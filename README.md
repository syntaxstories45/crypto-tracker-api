# Crypto Transaction Analyzer 🔍💱

A cryptocurrency transaction analyzer built using Python + Flask during a hackathon.

## 🚀 Features
- Analyze Bitcoin/Ethereum transactions  
- Fetch real-time transaction status  
- Display amount, sender, receiver, confirmations  
- Simple and clean UI  
- Flask backend running on localhost  

## 🧠 Tech Stack
- Python
- Flask
- Requests
- HTML/CSS
- Blockchain API (BlockCypher / Etherscan)
